# キャリブレーション機能について (About the Calibration Feature)

このディレクトリの機能実装には、Unityアセットストアの有料アセットである「**FinalIK**」を利用しています。

アセットのライセンス規約を尊重するため、FinalIKのAPIに直接依存するソースコードは、この公開リポジトリには含めておりません。
